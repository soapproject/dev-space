## Release 方案

只要在 GitHub 上啟用 `Actions`，然後執行 `Create Release` workflow，就能快速產生一個 Release。

操作步驟：
1. 前往 GitHub repository 的 `Actions` 分頁。
2. 選擇 `Create Release` workflow。
3. 點擊 `Run workflow`，輸入 `tag_name`、`release_name` 和 release notes。
4. 執行完成後即可在 `Releases` 畫面看到該 Release。
